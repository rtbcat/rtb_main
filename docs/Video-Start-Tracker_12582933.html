RTB Cat Help Wiki : Video-Start Tracker

Created by Andrii Perkhun, last modified by Jen Brannstrom on Jan 04, 2017

If you wish to optimise for video fill-rate then the RTBCat Tracker can track each video ad-start on advertiser's ad-tag. Once the RTBCat ad-start-tracker can see the video-ad-starts, your rules & ML will become active.

The following is needed to set this up.

How to add a video-start to the 'Ad-Start Tracker' 

To add a video-start to the tracker DB simply perform a GET call to this API end point:

GET

http://imp.rtb.cat/adstart/new?advid=${ADV_ID}&cpid=${CP_ID}&creativeid=${CREATIVE_ID}&auctionid=${AUCTION_ID}

Parameters

  • CpId - ${CP_ID} - advertiser ID in APPNX, should be put by advertiser (optional)
  • AdvId - ${ADV_ID} - campaign ID in APNX, should be put by advertiser (optional)
  • CreativeId - ${CREATIVE_ID} - creative ID in APNX, should be put by advertiser (optional)
  • AuctionId - ${AUCTION_ID} - auction ID - main parameter  (required)
  • CPVM - a bid for ad-start in other words this is payout for each video played in a ad banner (required)


Note:

For highest efficiency & best results you should make the GET requests as near-real-time as possible.
