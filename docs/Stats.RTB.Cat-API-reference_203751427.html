RTB Cat Help Wiki : Stats.RTB.Cat API reference

Created by Andrii Perkhun, last modified on Nov 28, 2017

List of the domain lists

GET https://stats.rtb.cat/api/v1/client/domain/

Header - { Authorization: Token <token>}

Response
Response: {
	domainList: [
		{
			id: {id}
			name: {name}
		},
		{
			id: {id}
			name: {name}
		}
		……
		]
		}

Response status 200


Create domain list

POST https://stats.rtb.cat/api/v1/client/domain/

Header - { Authorization: Token <token>}

Body
Body {
	name: {name},
	domains: {array}
	}


Response
Response: {
	id: {id},
}

Response status 200


Get domain list by id

GET https://stats-dev.rtb.cat/api/v1/client/domain/{domainId}/

Header - { Authorization: Token <token>}

Response
Response: {
	id: {id},
	name: {name},
	domains: [
				‘{domain}’,
				‘{domain}’,
				‘{domain}’,
	]
}

Response status 200


Update domain list by id

PUT https://stats.rtb.cat/api/v1/client/domain/{domainId}/

Header - { Authorization: Token <token>}


Body
Body {
	action: {append | remove | replace}
	domains: {array}
}


Response
Response: {
	id: {id},
	name: {name},
	domains: [
				‘{domain}’,
				‘{domain}’,
				‘{domain}’,
	]
}

Response status 200

Remove domain list by id

DELETE https://stats.rtb.cat/api/v1/client/domain/{domainId}/

Header - { Authorization: Token <token>}

Response status 200


Apply domain list to campaign

GET https://stats.rtb.cat/api/v1/client/domain/{domainId}/apply?&campaign_id={campaign_id}&action={ whitelist | blacklist | remove}

Header - { Authorization: Token <token>}

Response status 200

Campaigns list

GET https://stats-dev.rtb.cat/api/v1/client/campaign

Header - { Authorization: Token <token>}


Response
Response: {
	"campaignList": [
		{
			"id": {id},
			"name": {name},
			"state": {"active"/”inactive”}
		},
	]
}

Response status 200
