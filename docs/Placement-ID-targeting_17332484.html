RTB Cat Help Wiki : Placement ID targeting

Created by Jen Brannstrom, last modified on Jul 28, 2017

All exchange except Google AdX support placement ID's.

Placement ID's can be specific locations on a domain, or (in rare cases) they can be several domains. It all depends on the SSP and the publisher that owns the domain.

Some legacy domains still use one ad-tag (one placement ID) for a lot of websites.

RTB.Cat has no influence over this, it's simply what is made visible to us in the RTB auction.

To target specific placement ID's you must use:

 *:*:placement_id, eg:

*:*:3282767

(asterisk, dot, asterisk, colon, id)
