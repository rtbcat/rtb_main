When using LKQD ad server please use the following config.

if you want to buy from Google Ad Exchange

  1. all calls must be HTTPS !!
    This means every image, pixel, icon, logo, ....everything that gets called in the ad-chain must be https://. 

  2. please add the macro 
    [UNENCODED_CLICK_REDIRECT] or
    [ENCODED_CLICK_REDIRECT] or
    [DBL-ENCODED_CLICK_REDIRECT]

    VAST URL: place it like this:eg: http://ssp.lkqd.net/ad?pid=XXX&sid=123456&output=vastvpaid&support=html5flash&execution=any&placement=&playinit=auto&volume=100&width=[WIDTH]&height=[HEIGHT]&dnt=[DO_NOT_TRACK]&ip=[IP]&ua=[USERAGENT]&pageurl=[PAGE_URL]&contentid=[CONTENT_ID]&contenttitle=[CONTENT_TITLE]&contentlength=[VIDEO_DURATION]&contenturl=[CONTENT_URL]&rnd=[RANDOM_NUMBER]&[UNENCODED_CLICK_REDIRECT]

    in JS, like this:
     
    only one of the click macro's is needed.

  3. To make things easy for your optimisation, add some identifier macro's, eg: [STRATEGY_ID] or [CREATIVE_NAME] or [CREATIVE_ID]
