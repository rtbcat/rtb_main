RTB Cat Help Wiki : Rule-based optimisation

Created by Jen Brannstrom, last modified on Jun 27, 2018

rtb.cat supports basic rule-setting to blacklist/whitelisting of items.

The format is like this: for [targeting] if [metric [<, >, ==] number] in [day] then [action]


example 1:

for site_id if cpa > 3 in last 3 days then blacklist


example 2:

for pub_id if cpa > 1.5 and accepted_clicks > 100 in yesterday then whitelist


example 3:

for placement if impressions = 1000 and accepted_clicks < 5 in today then blacklist


Available parameters to use are:

for [select targeting option: site_id, pub_id, domain, placement]

if [select metric: impressions, accepted_clicks, conversions, cpc, cpa, media_cost, roi]

operators are: <, >, == 

[input: number]

in [select: today, yesterday, last n days (where n≤30 and n≠0)]

then [blacklist / whitelist]
