RTB Cat Help Wiki : Conversion Tracker

Created by Andrii Perkhun, last modified on Jun 08, 2017

RTBCat Tracker can track each conversion on advertiser's ad tag. This is needed to supply RTBCat stats and ML algorithms with data.

Add conversion tracker on advertiser's page 

  1. Add html script tag on page where ads point to (both HTTP and HTTPS versions are available)


Conversion tracker js-URL pixel

<script src="http://imp.rtb.cat/conversiontr.js?advid=${ADV_ID}&delAfterConv=yes&isNeedUseCookieForConv=yes"></script>

Most parameters are analogous to AppNexus's Creativer Macroses:

  • advid - ${ADV_ID} - advertiser ID in APPNX, should be put by advertiser (optional)
  • cpid - ${CP_ID} - campaign ID in APNX, should be put by advertiser (required)
  • CreativeId - ${CREATIVE_ID} - creative ID in APNX, should be put by advertiser (optional)

And additional parameter: 

  • delAfterConv - remove cookie after tracking ( yes / no )
  • isNeedUseCookieForConv - use conversion's cookie or not ( yes / no ). If yes it will use conversion cookie set by click tracker, otherwise - click cookie set by impression tracker

This pixel will add one cookie on users prowser to be used later by conversion tracker js-pixels. 

Direct add tracked conversion to Conversion Tracker 

To add conversion to Conversion Tracker DB simple GET call should be done to next API end point:

GET

http://imp.rtb.cat/conv/new?advid=${ADV_ID}&cpid=${CP_ID}&creativeid=${CREATIVE_ID}&auctionid=${AUCTION_ID}

Parameters

  • CpId - ${CP_ID} - advertiser ID in APPNX, should be put by advertiser (optional)
  • AdvId - ${ADV_ID} - campaign ID in APNX, should be put by advertiser (optional)
  • CreativeId - ${CREATIVE_ID} - creative ID in APNX, should be put by advertiser (optional)
  • AuctionId - ${AUCTION_ID} - auction ID - main parameter  (required)
