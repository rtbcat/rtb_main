RTB Cat Help Wiki : First Campaign setup

Created by Jen Brannstrom, last modified on Dec 20, 2016

Dear media Buyer, I know you know how to use a DSP :)
This will just help you get a "feel" for how AppNexus responds.


  1. Conversion Pixels:
    - create a conversion pixel
    EXPORT Pixel

  2. Line Item (aka Campaign group): Create a line item
    Basic Setup:
    - "do not track revenue" (trust me, just select it)
    - "Budget": unlimited (again, trust me. All will be clear in a moment)
    Optimisation:
    Frequency Cap
    - Leave empty
    Targeting:
    - do NOT set any targeting
    Associated creatives:
    - Select: "Manage creatives on the Campaign level"
    SAVE

  3. Campaign:

    - create a New Campaign, select the line item
    - Select: "Buy direct/third-party inventory"
    Basic Setup:
    - Name it
    - Budget: Media Cost: lifetime $1000, Daily: $100
    - "Pay on a Per-impression (CPM) basis"
    Targeting:
    - Switch off "Apps Placements" (Apps should be separate campaigns)
    - "Inventory Quality" set to "Serve on any inventory"
    - Inventory Targeting: http://prntscr.com/c8ke1w

    Select "3rd Party inventory" & target "Appnexus Direct"

    - Geo: as needed
    - Frequency: as needed
    - System: as needed
    - Demographics: the demo targeting is mediocre quality. It will most likely just cut traffic. Demographic data will be covered later.
    Associated creatives:
    - ONLY add the landing-page/tracking URL
    SAVE

  4. Creative Manager:
    - upload at least one simple creative, eg: 300x250
    - select: "Set landing page at the line item or campaign level."
    - Optional: 3rd Party impression pixel
    - Important: under "Creative Attributes" you must select "Platform Audit" (you will pay $1). Also set the language.
    - NEXT, Select " This creative is hosted on AppNexus and makes no other ad calls."
    - NEXT, 'Creative Asscoiation' Select the Campaign Tab & Campaign Name

  5. Segment:
    - Create Segment
    segment type: Pixel
    Keep user in segment: indefinitely
    Remarketing: tick the box (Some of the biggest clients on APNX make their profit solely with Retargeting segments. Without the retaregting they break-even.)

  6. Wait: It can take up to 15 min before all the settings have propagated. Not including ad-approvals.
