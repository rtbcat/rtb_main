RTB Cat Help Wiki : Add Impression Tracker to APNX advertiser

Created by Andrii Perkhun, last modified by Jen Brannstrom on Apr 24, 2018

RTBCat Tracker can track each impression served by AppNexus. This is needed to supply RTBCat stats and ML algorithms with data.

Step-by-step guide

  1. In AppNexus console UI go to advertiser third-party pixels list here http://console.appnexus.com/buyside/advertiser:YOUR ADV ID HERE/third-party-pixel
  2. Add a pixel like shown on screenshot:


Impression tracker js-URL pixel

http://imp.rtb.cat/imptr.js?advid=${ADV_ID}&tid=${TAG_ID}&uid=${USER_ID}&uc=${USER_CITY}&ucy=${USER_COUNTRY}&us=${USER_STATE}&sf=${SESSION_FREQ}&ru=${REM_USER}&sids=${SEG_IDS}&sc=${SEG_CODES}&pp=${PRICE_PAID}&rp=${RESERVE_PRICE}&ecp=${ECP}&cmid=${CUSTOM_MODEL_ID}&cmtm=${CUSTOM_MODEL_LAST_MODIFIED}&cmln=${CUSTOM_MODEL_LEAF_NAME}&cpid=${CP_ID}&cpgid=${CPG_ID}&cb=${CACHEBUSTER}&bp=${BID_PRICE}&aid=${AUCTION_ID}&age=${AGE}&advf=${ADV_FREQ}&creativeid=${CREATIVE_ID}&timeforcookie=1000


Impression tracker js-URL pixel (HTTPS version)

https://imp.rtb.cat/imptr.js?advid=${ADV_ID}&tid=${TAG_ID}&uid=${USER_ID}&uc=${USER_CITY}&ucy=${USER_COUNTRY}&us=${USER_STATE}&sf=${SESSION_FREQ}&ru=${REM_USER}&sids=${SEG_IDS}&sc=${SEG_CODES}&pp=${PRICE_PAID}&rp=${RESERVE_PRICE}&ecp=${ECP}&cmid=${CUSTOM_MODEL_ID}&cmtm=${CUSTOM_MODEL_LAST_MODIFIED}&cmln=${CUSTOM_MODEL_LEAF_NAME}&cpid=${CP_ID}&cpgid=${CPG_ID}&cb=${CACHEBUSTER}&bp=${BID_PRICE}&aid=${AUCTION_ID}&age=${AGE}&advf=${ADV_FREQ}&creativeid=${CREATIVE_ID}&timeforcookie=1000

Most parameters are AppNexus's Creativer Macros:

  • tid - {TAG_ID}
  • uid - {USER_ID}
  • uc - {USER_CITY}
  • ucy - {USER_COUNTRY}
  • us - {USER_STATE}
  • sf - {SESSION_FREQ}
  • ru - {REM_USER}
  • sids - {SEG_IDS}
  • sc - {SEG_CODES}
  • pp - {PRICE_PAID}
  • rp - {RESERVE_PRICE}
  • ecp - {ECP}
  • cmid - {CUSTOM_MODEL_ID}
  • cmtm - {CUSTOM_MODEL_LAST_MODIFIED}
  • cmln - {CUSTOM_MODEL_LEAF_NAME}
  • cpid - {CP_ID}
  • cpgid - {CPG_ID}
  • cb - {CACHEBUSTER}
  • bp - {BID_PRICE}
  • aid - {AUCTION_ID}
  • age - {AGE}
  • advf - {ADV_FREQ}
  • CreativeId - ${CREATIVE_ID}
  • AuctionId - ${AUCTION_ID}

And additional parameter: 

  • timeforcookie - cookie's time to live in minutes


This pixel will add one cookie on users browser to be used later by click tracker and conversion tracker js-pixels. Also it will try to get the origin of publisher's page (domain) and send it to our tracker with all impression's data.
