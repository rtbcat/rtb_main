RTB Cat Help Wiki : Click Tracker

Created by Andrii Perkhun, last modified on Jun 08, 2017

RTBCat Tracker can track each click on advertiser's ad tag. This is needed to supply RTBCat stats and ML algorithms with data.

Add click tracker on advertiser's page 

  1. Add html script tag on page where ads point to (both HTTP and HTTPS versions are available)


Click tracker js-URL pixel

<script src="http://imp.rtb.cat/clictr.js?advid=${ADV_ID}&delAfterClick=yes&isNeedCookieForConv=yes"></script>

Most parameters are analogous to AppNexus's Creativer Macroses:

  • advid - ${ADV_ID} - advertiser ID in APPNX, should be put by advertiser (optional)

And additional parameter: 

  • delAfterClick - remove cookie after tracking ( yes / no )
  • isNeedCookieForConv - create new cookie for conversion after click ( yes / no )

This pixel will add one cookie on users browser to be used later by conversion tracker js-pixels. 

Direct add tracked click to Click Tracker 

To add click to Click Tracker DB simple GET call should be done to next API end point:

GET

http://imp.rtb.cat/ct/new?advid=${ADV_ID}&cpid=${CP_ID}&creativeid=${CREATIVE_ID}&auctionid=${AUCTION_ID}

Parameters

  • CpId - ${CP_ID} - advertiser ID in APPNX, should be put by advertiser (optional)
  • AdvId - ${ADV_ID} - campaign ID in APNX, should be put by advertiser (optional)
  • CreativeId - ${CREATIVE_ID} - creative ID in APNX, should be put by advertiser (optional)
  • AuctionId - ${AUCTION_ID} - auction ID - main parameter  (required)
