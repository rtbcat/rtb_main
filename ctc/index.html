Click-to-Call ads

call-centre lead-gen

This is an example of a "click-to-call ad" with video.
It runs on display. Networks available: Google ADX, Rubicon, AOL, MoPub, AppNexus (msn.com), etc.

Name Click-to-Call ads
FormatCan be 30 sec MP4 video file or plain static image.
On-click the phone will open up the dialler and pre-populate the number to your preconfigured telephone number
Sizes300x250 & 300x600
price range$0.82-$5.72 CPM
Device TargetingWeb: Mobile & mobile in-app - Click-to-call is NOT SUITABLE FOR DESKTOP
GEO TargetingCountry, City, DMA, Zip-code
TrackingAdd your dedicated phone-number for call-tracking. Cookies are set on mobile websites. Google Ad ID's are collected for in-app.
Optionally: add tracking for Voluum, Appsflyer, GA, iMobiTrax, Thrive, etc.
Pixel-fire or postback, both work fine.
Re-targeting If you can pass back conversions from your call-tracking platform, then we can also set up call-reminders for those that clicked, but didn't complete the call.
Custom http webhooks or postbacks can be set up.
LimitationsWorks on Chrome browser for Android
Users that have opted-in to use Chrome's "Data Saver" will only see the static image, the video wil be blocked. The click-to-call button remains though.

By default Audio is OFF. User must initiate audio.

Safari on iPhones is in beta
